DDoS Attack tool.

Requirements
----
* [Python 3.9.4](https://www.python.org)

Modules
----
    pip3 install socket
    pip3 install colorama
    
Installation
----
    git clone https://github.com/haithamaouati/script.git
    cd DDoS
    chmod+x DDoS.py
    
Usage
----
    python3 DDoS.py -i <ip> -p <port>
    
Help
----
    python3 DDoS.py -h
    
Platform
----
* Windows
* Linux
* macOS
